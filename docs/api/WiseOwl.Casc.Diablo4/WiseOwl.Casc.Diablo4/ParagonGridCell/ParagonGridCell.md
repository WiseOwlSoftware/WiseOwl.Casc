# ParagonGridCell constructor

A grid coordinate on a paragon board's authored layout — row-major from the top-left (row `0` = top, col `0` = left). Paired with a [`ParagonNodeInfo`](../ParagonNodeInfo.md) by `Catalog.GetBoardNodes(int)` to describe placement on the board.

public ParagonGridCell(int Row, int Col)

| parameter | description |
| --- | --- |
| Row | Row index (`0` = top); always in `[0, board.Width)`. |
| Col | Column index (`0` = left); always in `[0, board.Width)`. |
