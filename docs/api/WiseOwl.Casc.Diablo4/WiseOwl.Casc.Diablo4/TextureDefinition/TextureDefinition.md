# TextureDefinition constructor

A parsed Diablo IV `TextureDefinition`: pixel format, base dimensions, the per-mip `serTex` table, and the atlas `ptFrame` sub-rects. The pixel payload itself is fetched separately (it is addressable by SNO id); this metadata only lives in the combined bundle (see [`CombinedTextureMeta`](../CombinedTextureMeta.md)).

```csharp
public TextureDefinition(int Format, int Width, int Height, int MipMin, int MipMax, 
    IReadOnlyList<SerialDataInfo> SerTex, IReadOnlyList<TexFrame> Frames)
```

| parameter | description |
| --- | --- |
| Format | Raw `eTexFormat` selector value. |
| Width | Base mip width in pixels. |
| Height | Base mip height in pixels. |
| MipMin | First valid mip level. |
| MipMax | Last valid mip level. |
| SerTex | Per-mip data locators (index 0 = largest mip). |
| Frames | Atlas sub-rects, keyed by image handle. |
