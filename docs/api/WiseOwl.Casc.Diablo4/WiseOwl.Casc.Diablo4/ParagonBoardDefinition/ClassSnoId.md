# ParagonBoardDefinition.ClassSnoId property

The owning class's PlayerClass SNO id — the stable per-class key (FR-D1), or `0` if this instance was decoded via the byte-only [`Parse`](./Parse.md) overload (which has no [`CoreToc`](../CoreToc.md) to resolve the name convention).

## Remarks

A `ParagonBoard` record carries no class field; the only first-party source is the SNO-name convention `Paragon_<ClassToken>_<Index>`. Per the durable opaque-id principle (`docs/casc-diablo4-format.md` Appendix C) that convention is decoded once, library-side — the token is the unique case-sensitive prefix of exactly one PlayerClass roster SnoName (§6.6, CL-16) — and exposed typed here; it is never a consumer regex. Populated by [`ReadParagonBoard`](../Diablo4Storage/ReadParagonBoard.md).
