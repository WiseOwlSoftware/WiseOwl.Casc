# NodeActivation constructor

FR-C16 R11 — the typed activation condition of a recipe layer: the engine-sourced predicate over [`NodeFact`](../NodeFact.md) that gates whether the layer draws. The consumer supplies the set of currently-true facts and calls [`Evaluate`](./Evaluate.md); it authors no predicate of its own.
