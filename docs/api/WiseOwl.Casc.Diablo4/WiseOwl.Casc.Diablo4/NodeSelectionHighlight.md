# NodeSelectionHighlight structure

FR-C19 вЂ” the paragon node mouse-over (hover) selection highlight, as the engine authors it: the `ContextualHighlight_Square` recipe (a 4-piece `TiledWindowPieces` TiledStyle вЂ” the named "square contextual highlight") paired with its drawable corner art (the 4 corner frames of the `2DUITiled_SelectionHighlight` atlas, surfaced via `SelectionRectangleInset`'s window-pieces). Drawing recipe (authored, owner-validated): a hollow square border sized to the node's bounding square. Draw only the 4 corners вЂ” each piece is a full quadrant (corner + both half-edges), so the four meet to surround the node; place [`TopLeft`](./NodeSelectionHighlight/TopLeft.md) in the top-left quadrant, [`TopRight`](./NodeSelectionHighlight/TopRight.md) top-right, etc. No edge or centre pieces, no fill (the node shows through).
