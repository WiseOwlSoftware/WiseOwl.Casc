# DerivedStat enumeration

A Character-Sheet stat that a [`CoreStat`](./CoreStat.md) contributes to via a per-point conversion. The four "signature" stats (Armor/ResistanceAllElements/HealingReceived/DodgeChance) are bound to a fixed core for every class; the three "mobile" stats (SkillDamage/CriticalStrikeChance/ResourceGeneration) are bound to a per-class core named by the class record (see [`StatConversions`](./PlayerClassDefinition/StatConversions.md)).
