## Remarks

Implements the layout in the upstream record (`d4-binary-formats.md §1`), re-derived here as a clean-room parser. All integers are little-endian; string pointers are resolved to absolute offsets. The whole file (~40 MB) is parsed from a single buffer using spans.

New-format header. The first Int32 is read as the group count; if it equals the sentinel `0xBCDE6611` the file is the current "new format" and the real count follows. The stock `WoW-Tools CascLib` NuGet (1.0.23) overflows on this header — this parser handles it directly.
