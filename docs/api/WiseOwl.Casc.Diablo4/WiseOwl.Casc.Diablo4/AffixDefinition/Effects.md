# AffixDefinition.Effects property

CL-92 (LIB-3) — the attribute(s) this affix modifies, one [`AffixEffect`](../AffixEffect.md) per `arModifiers` entry (a single-stat affix has one; a dual affix such as a two-element resistance has one per element). Each carries the `eAttribute` id + parameter and, after a full [`ReadAffix`](../Diablo4Storage/ReadAffix.md), the resolved localized [`AttributeName`](../AffixEffect/AttributeName.md). Empty (never `null`) when the affix authors no modifier array or the descriptor is malformed. See [`AffixEffect`](../AffixEffect.md) for the byte layout and the magnitude/operation boundary.
