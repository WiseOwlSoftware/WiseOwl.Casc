# AffixDefinition.StaticValues property

CL-94 — the affix's `"Static Value N"` scalars: the fixed, non-rolled numbers set/mythic/unique-power affixes carry (decoded from the `DT_VARIABLEARRAY[float]` at payload `+0xC0`), indexed positionally to match the affix [`Description`](./Description.md)'s `[Affix."Static Value N"]` placeholders. Each entry is a distinct quantity (a threshold, duration, percent, …), not a min/max of one stat — e.g. a Barbarian set power's `[100 Fury, 50%, 20%, 120%]`. Empty (never `null`) for the majority of affixes, whose magnitudes are the item-power formula on each [`FormulaGbid`](../AffixEffect/FormulaGbid.md) instead. This is the reliable in-record float field; the rolled per-item-power ranges are not literals here (see [`FormulaGbid`](../AffixEffect/FormulaGbid.md)).
