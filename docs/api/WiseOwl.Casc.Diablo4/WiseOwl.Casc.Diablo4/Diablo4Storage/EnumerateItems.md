# Diablo4Storage.EnumerateItems method

Enumerate every item (group Item = 73) whose base type falls in *category* — e.g. `EnumerateItems(ItemClass.Weapon)` for every weapon in the game, or `ItemClass.Charm` for every charm (§13 / LIB-1). Yields identity only ([`SnoId`](../ItemDefinition/SnoId.md) + [`ItemTypeSnoId`](../ItemDefinition/ItemTypeSnoId.md)); call [`ReadItem`](./ReadItem.md) for localized text on a chosen item. This is a full group scan (base-type classifications are memoized across the enumeration).
