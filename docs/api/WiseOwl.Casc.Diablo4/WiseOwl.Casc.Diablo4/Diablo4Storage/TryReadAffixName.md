# Diablo4Storage.TryReadAffixName method

Resolve an affix's localized display name without a full [`ReadAffix`](./ReadAffix.md) decode — the affix analogue of [`TryReadParagonBoardName`](./TryReadParagonBoardName.md). The name lives in the affix's sibling StringList table (group StringList = 42), CoreTOC name `"Affix_" + affixSnoName`, under label `Name` (§11.3 / CL-87). Strictly name-keyed via [`CoreToc`](./CoreToc.md) — the SNO ids are unrelated. Raw decoded value only, no fallback: returns the authored fragment verbatim (e.g. `"of Limitless Rage"`) with any `"Aspect …"` composition left to the consumer. Only ~1 in 4 group-104 affixes carry a `Name`; the remainder are unnamed system/internal affixes → honest `false`.

```csharp
public bool TryReadAffixName(int affixSnoId, out string name, string locale = "enUS")
```

| parameter | description |
| --- | --- |
| affixSnoId | The `Affix` SNO id (group 104). |
| name | The localized affix name, or Empty. |
| locale | Locale (default [`DefaultLocale`](./DefaultLocale.md)). |

## Return Value

`true` iff a localized name was decoded.
