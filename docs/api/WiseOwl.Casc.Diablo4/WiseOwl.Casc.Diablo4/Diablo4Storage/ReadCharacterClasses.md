## Remarks

Decoded clean-room (§6.5 / CL-17): the roster is PlayerClass (74); a group-74 entry is a real playable class iff the `General` StringList table (SNO GeneralStringTableSno) has label `"PlayerClass" + SnoName + "Male"` — this data-driven membership test excludes non-class junk (e.g. `Axe Bad Data`) with no hardcoded list. [`DisplayName`](../CharacterClass/DisplayName.md) is that label's localized value (the markup-free display form). Ordered by [`SnoId`](../CharacterClass/SnoId.md) ascending for determinism; the SnoId is the stable per-class key (never an array position), so the list survives a class being added/reordered next season. Cached per locale. Raw decoded values only — no policy/imaging.
