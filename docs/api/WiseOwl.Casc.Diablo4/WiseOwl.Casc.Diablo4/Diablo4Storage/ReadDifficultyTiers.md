# Diablo4Storage.ReadDifficultyTiers method

Read + decode the [`DifficultyTiersTable`](../DifficultyTiersTable.md) (FR-C34, CL-101) — the per-monster-level scaling curve (SNO [`DefaultSnoId`](../DifficultyTiersTable/DefaultSnoId.md) = 1973217, group GameBalance): 150 rows (monster levels 1..150) with per-level HP/damage multipliers, the XP-value anchor column, and a candidate gold column, plus the raw coefficient vector. This is the monster/content analogue of [`ReadLevelScaling`](./ReadLevelScaling.md) — a separate, far steeper curve (see the type remarks / §8.2).
