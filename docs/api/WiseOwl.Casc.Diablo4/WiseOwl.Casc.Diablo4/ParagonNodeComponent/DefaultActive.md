# ParagonNodeComponent.DefaultActive property

The layer's authored `bActive` — its default (resting) visibility. `true` (or unbound) ⇒ shown in the default/unselected state; `false` ⇒ default-off, shown only when the engine toggles it for a runtime state (selected, purchased, located, …). This is the field that disambiguates a swap pair: the `bActive=1` disc is the unselected variant, the `bActive=0` disc the selected one.
