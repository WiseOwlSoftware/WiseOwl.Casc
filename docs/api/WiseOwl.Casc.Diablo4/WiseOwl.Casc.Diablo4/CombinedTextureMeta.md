# CombinedTextureMeta class

The Diablo IV combined-meta bundle (magic `0x44CF00F5`, `Base\Texture-Base-Global.dat`): an indexed blob holding every [`TextureDefinition`](./TextureDefinition.md). Texture metadata is not path-addressable per-SNO; it is consolidated here, while the pixel payloads remain addressable by SNO id (de-duplicated through the shared-payload mapping).

## Remarks

Layout (clean-room, cross-checked against the upstream record `§8.12–§8.13`): `u32 magic; u32 count; count × { i32 snoId, u32 defSize }`. Then, per index entry in order, `descStart = alignUp8(prevEnd) + 8` (the `+8` is the Texture-group convention); the per-entry `snoId` is `i32 @descStart` and the [`TextureDefinition`](./TextureDefinition.md) body follows with payload base `descStart + 4`.
