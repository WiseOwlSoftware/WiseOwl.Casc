# MonsterLevelCurvePoint constructor

One row of a [`MonsterLevelCurve`](../MonsterLevelCurve.md) — a 12-byte record (two `int32` + one `float32`). [`Level`](./Level.md)/[`LevelHigh`](./LevelHigh.md) are the (equal, in live data) monster level and [`ScaledValue`](./ScaledValue.md) the scaled effective value; the precise remap semantic is a structural inference (see [`MonsterLevelCurvesTable`](../MonsterLevelCurvesTable.md) remarks) — the raw fields are all exposed.

```csharp
public MonsterLevelCurvePoint(int Level, int LevelHigh, float ScaledValue)
```

| parameter | description |
| --- | --- |
| Level | First `int32` — the monster/area level (row +0). |
| LevelHigh | Second `int32` — equal to [`Level`](./Level.md) in the live data; the upper bound of the level band if it is one (row +4). |
| ScaledValue | The `float32` scaled value (row +8) — climbs to `100` across the tier's level span. |
