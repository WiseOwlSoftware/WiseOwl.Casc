# PowerDefinition.Modifiers property

LIB-5 (CL-104) — the skill's selectable modifiers (the skill-tree enhancement / upgrade nodes: each active skill offers several, chosen in mutually-exclusive groups). Each carries its localized display [`Name`](../PowerModifier/Name.md) and the full effect [`Description`](../PowerModifier/Description.md) (with D4 markup/substitution tokens, like [`Description`](./Description.md)). Decoded from the power's sibling StringList (`Power_<snoName>`) `Mod<N>_Name` / `Mod<N>_Description` labels (the generalized §6.7 sibling convention). Ordered by the modifier index [`Index`](../PowerModifier/Index.md) (which is sparse — the engine skips unused slots). Empty for powers with no modifiers (passives, non-skill powers). The modifier group assignment (which choices are mutually exclusive) is not yet decoded — a follow-up; the names + effect text are complete.
