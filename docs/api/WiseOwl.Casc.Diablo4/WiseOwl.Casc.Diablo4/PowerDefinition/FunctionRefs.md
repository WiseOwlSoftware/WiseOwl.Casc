# PowerDefinition.FunctionRefs property

FR-C13 Phase 2 — engine-function references the power's formulas + localized [`Description`](./Description.md) rely on. The library surfaces each call (name + resolved-arg values) structurally; the consumer registers a per-name resolver to substitute the engine-runtime value (player-state accessors like `PlayerHealthMax()` are outside CASC's domain). Empty for the typical legendary passive (no function calls in the format string); non-empty for powers like Barbarian `Warbringer` whose format uses `[SF_1 * PlayerHealthMax()]`.
