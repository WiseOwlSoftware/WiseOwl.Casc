# SkillTree class

A decoded Diablo IV per-class skill tree (#57, CL-111) — the logical per-node structure the game builds from `SkillTreeRewards` (g20/547685) and the class board (g39). Exposes each node's kind, the skill it is/modifies, and its mutually-exclusive modifier group; the class's active skills; and a modifier-group membership lookup.

## Remarks

This is the logical tree (per the consumer's shape input on #57): what each node is, what it modifies, and the "pick one per group" rule. The visual graph — X/Y layout positions, edges, connector routing — lives in the g39 board record and is intentionally not surfaced here (no renderer consumer). A node's effect text is resolved through its [`SkillSno`](./SkillTreeNode/SkillSno.md): read that Power ([`ReadPower`](./Diablo4Storage/ReadPower.md)) for the skill's description and its [`Modifiers`](./PowerDefinition/Modifiers.md) (the modifier name/description list).

Scope (v1). Node kind, skill, and modifier group are decoded from `SkillTreeRewards` (byte-verified). The tier/category cluster grouping (Basic / Core / … / Ultimate) is derived from the g39 board topology and is a follow-up — see the #57 thread.
