## Remarks

Recovered clean-room (see `docs/casc-diablo4-format.md §6.5`, Appendix A CL-17): the roster is SNO group PlayerClass (74); the localized display name is the `General` StringList table (SNO 4118) under label `"PlayerClass" + SnoName + "Male"` (the markup-free display form; the gendered variants are identical strings, while the base `PlayerClass<SnoName>` label carries D4 `|5sing:plur` pluralization markup). A group-74 entry is a real playable class iff that label exists — this is the data-driven filter that excludes non-class junk entries (e.g. `Axe Bad Data`) with no hardcoded exclusion list.
