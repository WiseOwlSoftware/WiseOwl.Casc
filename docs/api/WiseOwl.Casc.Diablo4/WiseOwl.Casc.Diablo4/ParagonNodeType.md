# ParagonNodeType enumeration

The verified first-party paragon node `eNodeType` values ([`NodeType`](./ParagonNodeDefinition/NodeType.md), payload `+16`). A named-int convenience over the raw int — not policy; the raw int remains the serialized contract. Observed across all seven class boards: every class start node is Start (5); magic nodes are Magic (3); normal/structural/gate/rare nodes are Normal (0). This is a distinct axis from [`ParagonRarity`](./ParagonRarity.md) (e.g. a rare node is `NodeType 0` with `RarityOverride 3`); values other than 0/3/5 have not been observed and would surface here as the raw cast.
