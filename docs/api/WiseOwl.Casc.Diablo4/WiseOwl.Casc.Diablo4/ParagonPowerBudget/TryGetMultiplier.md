# ParagonPowerBudget.TryGetMultiplier method

Look up a budget-multiplier intrinsic by its canonical engine name (case-sensitive). The names follow the engine's formula-DSL convention: `ParagonPowerBudgetMultiplierNode<Rarity>{Major|Minor}<Off|Def>` — `NodeMagicDefensive`, `NodeRareMajorOffensive`, etc. (six total).

```csharp
public static bool TryGetMultiplier(string intrinsicName, out double value)
```

| parameter | description |

| value | The pinned multiplier on success. |

## Return Value
