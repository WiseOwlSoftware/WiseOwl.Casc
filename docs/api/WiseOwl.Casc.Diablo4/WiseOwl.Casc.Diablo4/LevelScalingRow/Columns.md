# LevelScalingRow.Columns property

Every column of the 212-byte row reinterpreted as Single (53 entries; index c = byte column `+4·c`). Only `Columns[1]` ([`HpScalar`](./HpScalar.md)) is a labeled, verified column; the rest are unlabeled per-level coefficients (see the type remarks — the Maxroll names are not asserted). Column 0 reads `0` in this table (the level is not stored per row — it is implied by row order; use [`Level`](./Level.md)); many trailing columns are `0` too.
