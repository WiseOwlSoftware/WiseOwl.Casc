# MonsterNameFragment constructor

One entry of the Diablo IV `MonsterNames` registry (FR-C35, CL-105) — a localized name fragment the game composes into an elite/special monster's displayed name (e.g. `FrozenSuffix004` → `"Frostburn"`, `ElectricLanceSuffix001` → `"Boltrend"`).

```csharp
public MonsterNameFragment(string Token, string Text, MonsterNameKind Kind)
```

| parameter | description |
| --- | --- |
| Token | The registry token / StringList label (e.g. `"FrozenSuffix004"`). Stable across builds; the composition key. |
| Text | The localized display fragment (e.g. `"Frostburn"`). |
| Kind | Whether the token is a Prefix or Suffix (classified from the token name; the game composes a full name from a base + prefix/suffix fragments). |
