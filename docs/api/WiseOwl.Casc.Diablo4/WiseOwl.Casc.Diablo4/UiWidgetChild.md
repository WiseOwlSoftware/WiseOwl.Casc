# UiWidgetChild record

An anonymous child sub-record of a [`UiWidget`](./UiWidget.md) (FR-C16 R9 / FR-C18). The Diablo IV UI scene serializes a composite node template (e.g. `Template_Node_Magic`) as a parent widget whose disc / ornate / overlay layers are name-less nested records — each a class id + `0xFFFFFFFF` sentinel followed by its own schema + positionally-keyed value records, structurally identical to a named widget minus the inline name. Decoding them structurally keeps each layer's authored rect paired with its `hImageFrame` handle.
