## Remarks

The record is a `TiledStyleDefinition` SNO whose first (and only, in every record observed) polymorphic `ptWindowPiece` element is decoded. The element's PolymorphicBase header puts the variant tag (`dwType`) at blob `+0x50`; the `NSlice` struct fields then map as `struct +N → blob +0x48+N` (so `flImageScale` @ struct +0x10 → blob +0x58, `hSourceImage` @ +0x18 → blob +0x60, `fTileCenter` @ +0x40 → blob +0x88, etc.). The two intervening `DT_VARIABLEARRAY` fields (struct +0x20 / +0x30) are 16-byte descriptors. Only the [`TypeTagNSlice`](./TypeTagNSlice.md) variant's tile-flag suffix is decoded; other variants set the tile flags to `-1` and [`HasPartialDecode`](./HasPartialDecode.md) to `true`.
