# DecodedImage constructor

A decoded image: tightly-packed, row-major, straight-alpha RGBA32 (`R,G,B,A` bytes). Deliberately not tied to any imaging library — the consumer owns cropping/encoding/compositing (slice with [`PixelRect`](../TexFrame/PixelRect.md)).


| Width | Pixel width. |
| Height | Pixel height. |
| Rgba | Pixel bytes, length `Width*Height*4`. |
