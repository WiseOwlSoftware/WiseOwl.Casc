# DifficultyTierRow.Columns property

Every column of the 128-byte row reinterpreted as Single (32 entries; index c = byte column `+4·c`). Caveats:`Columns[0]` is the level as an `int32` (use [`Level`](./Level.md)), and columns 3/4/5/6 (bytes `+12..+24`) are small `int` flags, not floats — reinterpret those as `int` via Single). The labeled columns (1/2/9/10) are the typed accessors above; the rest are unlabeled per-level reward/scaling coefficients, exposed raw so no data is hidden (comprehensive-data-exposure) without asserting names that can't be verified.
