# ParagonTooltipChrome.ItemSidePanelByRarityName property

Future-proofing handle on the four item-side rarity panels (Unique / Set / Mythic / Season). Keyed by the engine's string rarity token (the suffix from the `TooltipBackgroundRarity_*` SNO name — `"Unique"`, `"Set"`, `"Mythic"`, `"Season"`); not keyed by [`ParagonRarity`](../ParagonRarity.md) because none of these rarities apply to paragon nodes. Available for any item-tooltip work the consumer takes on.
