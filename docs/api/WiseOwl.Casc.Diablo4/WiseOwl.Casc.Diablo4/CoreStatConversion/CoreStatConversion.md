# CoreStatConversion constructor

One core-attribute → derived-stat conversion: how many units of [`Stat`](./Stat.md) each point of [`Core`](./Core.md) grants. The coefficient is a universal engine constant (identical for every class — only the core→stat mapping is per-class).

```csharp
public CoreStatConversion(CoreStat Core, DerivedStat Stat, double PerPoint, ConversionUnit Unit)
```

| parameter | description |
| --- | --- |
| Core | The source core attribute. |
| Stat | The derived stat it feeds. |
| PerPoint | Units of *Stat* per point of *Core* (interpret with *Unit*). |
| Unit | Whether *PerPoint* is flat or a percentage. |
