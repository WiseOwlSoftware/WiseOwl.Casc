# PowerModifier constructor

LIB-5 (CL-104) — one selectable modifier of a skill ([`Modifiers`](../PowerDefinition/Modifiers.md)): a skill-tree enhancement / upgrade node with its localized display name + full effect description.

```csharp
public PowerModifier(int Index, string Name, string Description)
```

| parameter | description |
| --- | --- |
| Index | The modifier's slot index (the `N` in the sibling StringList's `Mod<N>_Name` label). Sparse — the engine skips unused slots (e.g. Blade Shift uses 0,1,2,3,5,7,9). Stable per build; the slot's group membership (mutual exclusivity) is a separate, not-yet-decoded field. |
| Name | The modifier's localized display name (e.g. `"Grenade Shift"`), from label `Mod<Index>_Name`. |
| Description | The modifier's localized effect text, from label `Mod<Index>_Description` — carries D4 markup / substitution tokens (`{c_number}`, `{SF_n}`, `{payload:…}`, …), same as [`Description`](../PowerDefinition/Description.md). Empty if the description label is absent. |
