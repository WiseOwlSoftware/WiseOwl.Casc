# StatTagDefinition.ThresholdFormulaText property

The ASCII formula text whose evaluation yields the stat threshold the gated bonus requires. For paragon-node-referenced tags the expression binds `ParagonBoardEquipIndex` (`"760 + (455 * ParagonBoardEquipIndex)"` → `2125` when `EquipIndex == 3`); for glyph-referenced tags the text is typically a bare numeric constant (`"40"`). Empty when the descriptor is missing or out-of-range.
