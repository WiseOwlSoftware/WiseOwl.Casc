# SharedPayloadMapping class

Diablo IV's `CoreTOCSharedPayloadsMapping.dat` (magic `0xABBA0003`): a payload de-duplication table. When a SNO's own payload section is empty, its pixel/blob bytes physically live under a different ("shared") SNO; this maps the requesting id → the id that actually holds the payload. Texture atlases (e.g. the per-class paragon node icons) rely on this.
