## Remarks

[`SnoId`](./ItemDefinition/SnoId.md) is the binary field (payload `0`). The localized strings are resolved from the item's sibling StringList table (`docs/casc-diablo4-format.md §11.4`, Appendix A CL-22 / CL-20): group-42 SNO `"Item_" + snoName`, labels `Name` / `Flavor` / `TransmogName`. Each is empty (honest sentinel) when decoded byte-only, when the item has no sibling table, or when that particular label is absent; the consumer owns any fallback. Raw decoded text — D4 markup kept intact.
