# AffixEffect.DataAttributeOrdinal property

CL-92 — the `DataAttributes` (SNO `1907204`) table ordinal when [`IsDataDefinedAttribute`](./IsDataDefinedAttribute.md) is `true` (`AttributeId & 0x7FFFFFFF`); otherwise the [`AttributeId`](./AttributeId.md) itself (an engine attribute is already its own id). Provided so consumers can index the DataAttributes table without re-deriving the flag — and to make explicit that the id must never be `abs()`-ed (the two namespaces overlap numerically).
