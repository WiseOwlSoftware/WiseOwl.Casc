# AffixEffect.IsDataDefinedAttribute property

CL-92 — true when [`AttributeId`](./AttributeId.md) is a reference into the data-defined `DataAttributes` designer table (SNO `1907204`) rather than the runtime engine `eAttribute` registry. Selected by the high bit (`0x80000000`); the table ordinal is then [`DataAttributeOrdinal`](./DataAttributeOrdinal.md). These are the conditional/seasonal attributes (Berserking, kill-streak, per-power bonuses); they do not resolve through [`GetAttributeName`](../Diablo4Storage/GetAttributeName.md) (a disjoint namespace).
