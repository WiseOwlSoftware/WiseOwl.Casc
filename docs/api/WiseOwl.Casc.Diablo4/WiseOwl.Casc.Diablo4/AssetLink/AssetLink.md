# AssetLink constructor

FR-C20 P5 — one authored relationship edge from an asset to another (e.g. a board to one of its nodes, a node to its passive power, a glyph to an affix it grants). The [`Target`](./Target.md) is a full [`AssetRef`](../AssetRef.md), so traversal chains: [`Related`](../Catalog/Related.md) the target again. Only authored FK edges are surfaced — runtime relationships (e.g. which glyph a player has socketed into a node) are not links.
