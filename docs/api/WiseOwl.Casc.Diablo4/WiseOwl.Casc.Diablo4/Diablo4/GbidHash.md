# Diablo4.GbidHash method

The Diablo IV GBID hash: a case-insensitive DJB2 over the name's ASCII bytes (`h = 0; h = h*33 + tolower(c)`, unsigned 32-bit, stopping at the first NUL). This is the canonical hash D4 uses to reference shared values across GameBalance / affix / skill / formula tables.

```csharp
public static uint GbidHash(string name)
```

| parameter | description |
| --- | --- |
| name | The reference name (ASCII). |

## Remarks

Verified: `GbidHash("ParagonNodeCoreStat_Normal") == 0x42C16A1B` (upstream `d4-binary-formats.md §7.1`). Parsing the tables that these GBIDs key into is consumer domain logic (see the boundary in `docs/casc-diablo4-format.md` Appendix C); this hash is the one piece that is generic and stable enough to be the library's canonical home.
