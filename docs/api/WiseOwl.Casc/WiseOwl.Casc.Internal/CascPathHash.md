# CascPathHash class

The 64-bit path hash CASC/TVFS uses to key file paths. The underlying mixing function is Bob Jenkins' public-domain `lookup3` hash (the `hashlittle2` variant, 96 bits of internal state) — the algorithm every CASC implementation uses for path lookup. The type is named for what it does here, not after the algorithm's author; the lineage is credited in the docs and `NOTICE`.
