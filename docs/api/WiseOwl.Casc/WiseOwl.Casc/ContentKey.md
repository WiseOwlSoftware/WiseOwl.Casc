# ContentKey structure

A content key (CKey): the MD5 of a file's logical, fully-decoded bytes. The game's `root`/TVFS maps game-meaningful names to CKeys; the `encoding` table maps a CKey to one or more [`EncodingKey`](./EncodingKey.md)s.
