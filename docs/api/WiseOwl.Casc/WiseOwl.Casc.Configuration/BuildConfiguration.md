# BuildConfiguration class

A parsed CASC build configuration file (found at `Data/config/<k0k1>/<k2k3>/<buildKey>`). It is a simple `key = value [value …]` text file (`#` comments) naming the content/encoding keys of the storage's core manifests: the `encoding` table, the `root`, and the `vfs-root` / `vfs-N` TVFS manifests.
