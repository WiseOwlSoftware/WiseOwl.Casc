# BuildInfo class

The parsed `.build.info` file at the root of a local Blizzard installation. It is a pipe-delimited table: a header row of `Name!TYPE:len` column descriptors, then one row per installed build. WiseOwl.Casc reads the active row and exposes the few fields the transport needs (build/CDN config keys, version, tags).
