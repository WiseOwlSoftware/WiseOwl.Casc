# EncodingTable class

The CASC encoding table: maps a [`ContentKey`](../WiseOwl.Casc/ContentKey.md) (the hash of a file's logical bytes) to the [`EncodingKey`](../WiseOwl.Casc/EncodingKey.md)(s) of its stored, BLTE-encoded form(s). The table is itself stored as a BLTE blob; its own key comes from the build configuration.

## Remarks

Clean-room implementation of the TACT encoding layout: an `EN`-magic header, an ESpec string block, a CKey page index, then the CKey pages — 4 KiB chunks of `{ byte keyCount; uint40BE fileSize; CKey; keyCount × EKey }` records, zero-padded to the chunk boundary. We index CKey → first EKey (sufficient to open content); the EKey/ESpec section is skipped.
