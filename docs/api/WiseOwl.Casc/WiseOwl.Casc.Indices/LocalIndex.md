# LocalIndex class

The CASC local index: the 16 `Data/data/<bucket>*.idx` files that map a 9-byte [`EncodingKey`](../WiseOwl.Casc/EncodingKey.md) prefix to an [`ArchiveLocation`](./ArchiveLocation.md). Together they index every blob physically present in the local `data.NNN` archives.

## Remarks

Clean-room implementation of the v2/v7 local-index layout: a hashed header (`HeaderHashSize`, `HeaderHash`, then the sized header block), padded to a 16-byte boundary, then `EntriesSize` / `EntriesHash` and a packed array of 18-byte entries (9-byte key, 1-byte index-high, 4-byte big-endian index-low, 4-byte little-endian size). The archive number and offset are bit-packed: `archive = (high << 2) | (low >> 30)`, `offset = low & 0x3FFFFFFF`. The first key wins on duplicates.
