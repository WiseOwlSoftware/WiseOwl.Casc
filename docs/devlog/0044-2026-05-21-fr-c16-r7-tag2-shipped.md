# 0044 ‚Äî FR-C16 R7: tag-2 field-value reader shipped (CL-48)

*2026-05-21*

Ships the grammar crack from devlog 0043, after owner approval to revise
the delivered FR-C7/C11 decode surfaces. The `UiScene` parser now reads
the second value-record encoding (the 12-byte tag-2 block) as a field
value, not just the 56-byte `0x22` record.

## The change

`UiScene.Parse` Pass 2 now, per widget:

1. Computes `ownEnd` = the first nested anonymous child marker (a class
   id + `0xFFFFFFFF` sentinel at `+0x08`) after the widget's own header,
   or the span end. The schema + value scans are confined to
   `[from, ownEnd)` so a parent's fields never absorb its children's.
2. Reads value records of **either** shape ‚Äî `0x22` (56 B) or tag-2
   (`2, 0, value`; 12 B) ‚Äî in serialized order, **capped at the schema
   field count**. The field-value run precedes any trailing `0x58`
   layer-block, so the count cleanly bounds it.

The CL-47 ¬±4096 rect guard is **removed** ‚Äî the exact decode makes it
dead code.
