# 0012 — FR-D1: first-party localized ParagonBoard display name

*2026-05-17*

The optimizer asked the recipe-or-gap question (`fr-d1`): can a paragon
board's localized in-game name ("Pyrosis", "Dynamism", "Start", …) be
resolved first-party so the L2 glyph-allocation ranking can label
per-board, replacing its interim
`IsStart ? "<Class> Start" : "<Class> <SnoName>"` placeholder.

## Recon

`ParagonBoardDefinition` (group 108) genuinely carries no name field
(§7.1: only `snoId`, `nWidth`, `arEntries`). `SnoScan find
Paragon_Warlock` showed the structure immediately: alongside each
group-108 board `Paragon_<Class>_<NN>` sits a group-42 StringList SNO
named `ParagonBoard_Paragon_<Class>_<NN>`. A new `SnoScan stl <sno>
<locale>` command (recon-only) dumped table 2458673 →
`[Name] = Start`; 2458679 (Warlock_03) → `Dynamism` (enUS) /
`Dynamismus` (deDE); Warlock_04 → `Pyrosis`. The label is `Name`; each
sibling table holds one entry.

Critically: the SNO ids have **no fixed offset**. Warlock's table is
`boardSnoId − 1` but Sorcerer's is not (`Paragon_Sorc_00` 939773 →
`ParagonBoard_Paragon_Sorc_00` 1111181). Resolution is strictly
CoreTOC-name-keyed — recorded so we never regress to id arithmetic
(CL-15).

## Verdict & delivery — (B)

The convention is undocumented D4 data → our side of the boundary, so
this is an API gap, not a recipe to hand the consumer. Shipped the
minimal typed surface: `Diablo4Storage.TryReadParagonBoardName` (raw
value, no fallback policy — consumer owns the SnoName fallback) and the
throwing `ReadParagonBoardName`, plus `SnoGroup.StringList = 42` (now a
named group; still not per-SNO path-addressable). No new public type.

Spec: new `casc-diablo4-format.md §6.4` (convention + verified-anchor
table) and Appendix A `CL-15`. Integration test
`ReadParagonBoardName_resolves_localized_board_name` asserts the §4
verbatim probes against live `3.0.2.71886`; full suite green, 0
warnings; API docs regenerated. Consumer report:
`docs/fr-d1-response.md`.

Same amend-until-next-publish contract as FR-C7. Nothing further owed
unless the gated release ships or a seasonal build changes the Build
Key.
